[Pygame](https://github.com/pygame/pygame)是 Pete Shinners 在 [SDL（Simple DirectMedia Layer）](https://github.com/libsdl-org/SDL)基础上开发而来, **主要用来开发PC客户端游戏**

**Pygame的坐标系：**

- 原点在左上角(0,0)
- x轴水平向右，逐渐增加,反之减少
- y轴水平向下，逐渐增加，反之减少

## 先安装 pygame
```bash
pip install pygame
```
